# Hostel Management System

## Project Overview

The Hostel Management System is a Python-based desktop application developed using Tkinter and SQLite. It helps manage hostel-related activities such as student login, room and fee management, food menu viewing, and food rating.

## Features

* Student Login System
* Hostel Room Management
* Fee Payment Management
* Food Menu Display
* Food Rating and Feedback System
* SQLite Database Integration

## Technologies Used

* Python
* Tkinter
* SQLite
* Git
* GitHub

## Project Files

* main.py
* login.py
* service.py
* menu.py
* rating.py
* hostelroom_fee.py

## How to Run

1. Install Python.
2. Download or clone the repository.
3. Open the project folder.
4. Run:

python main.py
